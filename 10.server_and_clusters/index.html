    服务器基础知识¶

    什么是服务器？

    在我看来，大多数语境下服务器指的就是网络中的一台主机（Host）。只有在强调服务关系时，我们才将主机进一步区分为服务器（Server）和客户端（Client）。

    进入如阿里云控制台这样的管理界面，我们可以看到一台云服务器实例（instance）的基本信息。

    对于一台云服务器，我们主要关注它的这些信息：

    • 实例 ID：服务器的唯一标识
    • 配置：
      • CPU：一般关心核心数量，型号并不重要。
      • 内存
      • 网络带宽
      • 存储
    • 公网 IP：实例的公网 IP 地址，用于访问互联网。
    • 安全组或防火墙：用于控制服务器入站和出站流量。

    参考手册¶

    在服务器运维中，很多时候你需要一个地方查找资料，如各种 Modules、系统的各种 API。这里搜集了一些：

    • LKDDb: Linux Kernel Driver DataBase | what Linux drivers are for me?。搜集了各种驱动的信息，当你遇到硬件上的问题时常常可以在这里找到答案。
    • The Linux man-pages project。这里包含 Linux Kernel 和 C 库的各种文档。
    • The Linux Kernel documentation

    其他学习资料¶

    • Linux 服务器运维贯通课：Linux Server Course - System Configuration and Operation - YouTube 该视频可以说是把 5 年的 Linux 运维经验压缩到了一个 5 小时的视频中，非常值得一看。
    • 关于 Linux 系统
      • 鸟哥的 Linux 私房菜系列。但它基于 RedHat 系统，主要用于企业服务器。相信看到这里的你已经具备了前面章节的各种 Linux 技能，这时发行版不同对你来说已不是问题，你已经学会迁移知识到其他系统上去了。
      • 《The Linux Programming Interface》Michael Kerrisk：详细讲解了 Linux 系统调用和 C 库的实现，是非常详细的关于Linux 系统编程和原理的书。
